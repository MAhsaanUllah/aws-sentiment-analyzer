# 🤖 AWS Sentiment Analyzer

AI-powered sentiment analysis API built on AWS.

## Tech Stack
- AWS Lambda (Python)
- AWS API Gateway
- Serverless Architecture
- HTML/CSS/JavaScript (Frontend)

## Features
- Real-time sentiment analysis
- POSITIVE / NEGATIVE / NEUTRAL detection
- REST API deployed on AWS
- Clean web interface

## Live API
POST https://pi7aowsm33.execute-api.us-east-1.amazonaws.com/Production/analyze

## Architecture
User → HTML UI → AWS API Gateway → Lambda → Response
